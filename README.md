## 計算兩個矩形是否有交集 （要計算是否交集, 需滿足四個條件）

![](docs/images/rect_intersects.png)

- 矩形 1 的左邊緣(X)是否小於等於矩形 2 的右邊緣(MaxX)
- 矩形 1 的右邊緣(MaxX)是否大於等於矩形 2 的左邊緣(X)
- 矩形 1 的上邊緣(Y)是否小於等於矩形 2 的下邊緣(MaxY)
- 矩形 1 的下邊緣(MaxY)是否大於等於矩形 2 的上邊緣(Y)
